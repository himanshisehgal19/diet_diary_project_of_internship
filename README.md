# diet_diary_project_of_internship
## The main idea for this project was that each user can maintain his diet diary so as to analyze his/her diet. On the main page via an API all the food items that are present on the company's website will be displayed as shown below:
![image](https://user-images.githubusercontent.com/55012463/121366642-44e30b00-c957-11eb-90ad-0577cc55709c.png)

 ##  After clicking on click me User can select any food item either for his consumption or for his purchasing or he/she can check the details of the food item

![image](https://user-images.githubusercontent.com/55012463/121366696-4e6c7300-c957-11eb-8724-ad3aa825df0f.png)
 # Here as for example the user selects the shrikhand and consumes 1 unit After clicking on this consume button an api of getfood is called and it will directly send the details to transaction table
